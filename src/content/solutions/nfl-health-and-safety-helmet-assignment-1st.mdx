# NFL Health & Safety - Helmet Assignment - 1st Place (2021) - Pipeline Template

## Problem

Object tracking / 2D-to-3D registration solution template for NFL Health & Safety - Helmet Assignment - 1st Place (2021), modeled as a 5-stage deterministic pipeline: Object detection, Temporal tracking, Sensor data alignment, Bipartite matching, and downstream competition-specific post-processing.

## Approach

This template captures the ordered steps used by NFL Health & Safety - Helmet Assignment - 1st Place (2021): prepare the data, build task-specific representations, train or apply the main model, and convert outputs into the final competition submission.

## CDG Walkthrough

1. **Object detection** -- This stage run YOLOv5 to detect all helmets in the video frames (Ultralytics).
1. **Temporal tracking** -- This stage link bounding boxes across frames using DeepSORT based on intersection-over-union (IoU) and CNN embeddings.
1. **Sensor data alignment** -- This stage project tabular player sensor data (RFID x/y coordinates) onto the 2D camera view using camera calibration matrices (OpenCV).
1. **Bipartite matching** -- This stage map tracked helmet clusters to projected player RFID clusters using Iterative Closest Point (ICP) and the Hungarian algorithm (scipy.optimize.linear_sum_assignment).
1. **Post-processing** -- This stage fix fragmented tracks and impute missing player assignments via forward-fill.

## Key Insight

The load-bearing idea is the pipeline topology: Object detection creates the representation needed by Post-processing, so later stages can optimize the competition metric without solving raw-data ingestion from scratch.
