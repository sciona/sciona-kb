# Stanford Ribonanza RNA Folding - 1st Place (2023) - Pipeline Template

## Problem

Bioinformatics / Graph + Sequence solution template for Stanford Ribonanza RNA Folding - 1st Place (2023), modeled as a 5-stage deterministic pipeline: Base Pair Probability (BPP) extraction, Graph formatting, Graph Transformer Encoder, 1D CNN Encoder, and downstream competition-specific finalization.

## Approach

This template captures the ordered steps used by Stanford Ribonanza RNA Folding - 1st Place (2023): prepare the data, build task-specific representations, run the main modeling or search loop, and convert outputs into the final benchmark submission.

## CDG Walkthrough

1. **Base Pair Probability (BPP) extraction** -- This stage process RNA sequence strings through domain-specific physics simulators to generate structural probability matrices (ViennaRNA/EternaFold).
1. **Graph formatting** -- This stage map the RNA nucleotides to nodes and the BPP connections to edges (numpy).
1. **Graph Transformer Encoder** -- This stage pass the graph structure through a deep Graph Transformer, explicitly injecting the chemical structure as positional encodings (PyTorch).
1. **1D CNN Encoder** -- This stage process the raw string sequence in parallel using a 1D ConvNeXT architecture to capture linear dependencies (PyTorch).
1. **Dual-head output** -- This stage fuse the representations and predict 2 continuous reactivity targets per nucleotide along the sequence using L1 loss.

## Key Insight

The load-bearing idea is the pipeline topology: Base Pair Probability (BPP) extraction creates the representation needed by Dual-head output, so later stages optimize the metric without solving raw-data routing from scratch.
