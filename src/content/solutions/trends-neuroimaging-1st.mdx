# TReNDS Neuroimaging - 1st Place (2020) - Pipeline Template

## Problem

Tabular Regression (derived from 3D MRI) solution template for TReNDS Neuroimaging - 1st Place (2020), modeled as a 5-stage deterministic pipeline: Brain mask application, Independent Component Analysis, Ridge feature engineering, Feature concatenation, and downstream competition-specific post-processing.

## Approach

This template captures the ordered steps used by TReNDS Neuroimaging - 1st Place (2020): prepare the data, build task-specific representations, train or apply the main model, and convert outputs into the final competition submission.

## CDG Walkthrough

{/* <ScionaViewerPlaceholder cdgId="solution.kaggle.trends_neuroimaging_1st" readOnly={true} /> */}

1. **Brain mask application** -- This stage map functional MRI matrices back into their 3D spatial voxel coordinates (numpy/nibabel).
1. **Independent Component Analysis** -- This stage calculate cross-correlations (FNC) between brain spatial maps across subjects.
1. **Ridge feature engineering** -- This stage run thousands of individual Ridge regressions treating every variable pair as dependent/independent to extract coefficients as metaparameters (sklearn).
1. **Feature concatenation** -- This stage flatten raw FNC maps, ICA loadings, and Ridge coefficients into a 1D tabular vector per patient.
1. **Multi-head modeling** -- This stage train separate Support Vector Regressors and Nu-SVR models for each of the 5 target variables (sklearn).

## Key Insight

The load-bearing idea is the pipeline topology: Brain mask application creates the representation needed by Multi-head modeling, so later stages can optimize the competition metric without solving raw-data ingestion from scratch.

## Failure Modes

- Distribution shift breaks handcrafted thresholds, post-processing rules, or validation-derived weights.
- Missing metadata prevents faithful construction of candidates, groups, windows, or cross-validation folds.
- A weak first-stage recall step cannot be recovered by downstream rescoring or ensembling.
