# M5 Forecasting (Uncertainty) - 1st Place (2020) - Pipeline Template

## Problem

Probabilistic Hierarchical Time Series solution template for M5 Forecasting (Uncertainty) - 1st Place (2020), modeled as a 5-stage deterministic pipeline: Base Forecasting, Residual calculation, Probability distribution fitting, Quantile Generation, and downstream competition-specific finalization.

## Approach

This template captures the ordered steps used by M5 Forecasting (Uncertainty) - 1st Place (2020): prepare the data, build task-specific representations, run the main modeling or search loop, and convert outputs into the final benchmark submission.

1. **Base Forecasting** -- This stage generate point predictions for all items using the standard M5 Accuracy methodology (LightGBM with Tweedie loss) (LightGBM).
1. **Residual calculation** -- This stage calculate the historical errors (residuals) of the trained model over the validation period (pandas/numpy).
1. **Probability distribution fitting** -- This stage fit continuous mathematical distributions (e.g., Gaussian, Negative Binomial) over the grouped residuals to estimate the variance for each product category (scipy.stats).
1. **Quantile Generation** -- This stage sample the fitted distributions to calculate the exact 9 discrete target quantiles (e.g., 0.005, 0.50, 0.995) required by the competition (numpy).
1. **Hierarchical scaling** -- This stage scale the lower-level quantiles algorithmically so that the statistical spread of the lowest item perfectly aligns with the required variance of the top-level national aggregate (custom math / scipy).

## Key Insight

The load-bearing idea is the pipeline topology: Base Forecasting creates the representation needed by Hierarchical scaling, so later stages optimize the metric without solving raw-data routing from scratch.
