# OpenVaccine COVID-19 mRNA Vaccine Degradation - 1st Place (2020) - Pipeline Template

## Problem

Structured Sequence Prediction solution template for OpenVaccine COVID-19 mRNA Vaccine Degradation - 1st Place (2020), modeled as a 5-stage deterministic pipeline: Base Pair extraction, Tensor formatting, GCN Layer, RNN Layer, and downstream competition-specific finalization.

## Approach

This template captures the ordered steps used by OpenVaccine COVID-19 mRNA Vaccine Degradation - 1st Place (2020): prepare the data, build task-specific representations, run the main modeling or search loop, and convert outputs into the final benchmark submission.

## CDG Walkthrough

{/* <ScionaViewerPlaceholder cdgId="solution.kaggle.openvaccine_mrna_degradation_1st" readOnly={true} /> */}

1. **Base Pair extraction** -- This stage parse the raw mRNA string representations through the ViennaRNA package to extract the Base Pair Probability (BPP) connection matrices.
1. **Tensor formatting** -- This stage stack the 1D sequence and 2D BPP matrices into compatible PyTorch tensors (numpy).
1. **GCN Layer** -- This stage process the 2D BPP matrices through a Graph Convolutional Network (GCN) to allow structural nodes to pass chemical information to their bonded neighbors (PyTorch Geometric).
1. **RNN Layer** -- This stage process the 1D sequence strings through a Bidirectional GRU to capture the linear sequence context (PyTorch).
1. **Target-weighted Loss** -- This stage combine the embeddings and apply a specialized loss function during backpropagation that heavily penalizes errors on sequences with high signal-to-noise ratios, ignoring mathematically noisy lab readings.

## Key Insight

The load-bearing idea is the pipeline topology: Base Pair extraction creates the representation needed by Target-weighted Loss, so later stages optimize the metric without solving raw-data routing from scratch.
