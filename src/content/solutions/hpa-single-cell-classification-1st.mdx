# Human Protein Atlas - Single Cell Classification - 1st Place (2021) - Pipeline Template

## Problem

Weakly Supervised Vision / Instance Segmentation solution template for Human Protein Atlas - Single Cell Classification - 1st Place (2021), modeled as a 5-stage deterministic pipeline: Instance Segmentation, Crop Extraction, Image-Level Training, Multi-Instance Learning (MIL), and downstream competition-specific finalization.

## Approach

This template captures the ordered steps used by Human Protein Atlas - Single Cell Classification - 1st Place (2021): prepare the data, build task-specific representations, run the main modeling or search loop, and convert outputs into the final benchmark submission.

## CDG Walkthrough

{/* <ScionaViewerPlaceholder cdgId="solution.kaggle.hpa_single_cell_classification_1st" readOnly={true} /> */}

1. **Instance Segmentation** -- This stage utilize the pre-trained Cellpose algorithm on the 4-channel imagery (microtubules, nucleus, etc.) to generate precise masks for every individual cell in the image (Cellpose).
1. **Crop Extraction** -- This stage extract thousands of individual cell image crops based on the generated masks (numpy).
1. **Image-Level Training** -- This stage train an EfficientNet model on the full, uncropped images utilizing the image-level ground truth labels (since cell-level labels do not exist) (PyTorch).
1. **Multi-Instance Learning (MIL)** -- This stage extract the cell-level crops, pass them through the trained network, and use a specialized MIL pooling layer (e.g., max-pooling across cells) to map cell-level probabilities back to the image-level label.
1. **Label assignment** -- This stage assign the highest probability classes back to the individual cell instances for final scoring.

## Key Insight

The load-bearing idea is the pipeline topology: Instance Segmentation creates the representation needed by Label assignment, so later stages optimize the metric without solving raw-data routing from scratch.

- A weak first-stage recall, search, or representation step cannot be fully recovered by downstream rescoring or ensembling.
