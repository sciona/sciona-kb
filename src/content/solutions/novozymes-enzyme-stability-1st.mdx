# Novozymes Enzyme Stability Prediction - 1st Place (2023) - Pipeline Template

Bioinformatics / 3D Protein Graph solution template for Novozymes Enzyme Stability Prediction - 1st Place (2023), modeled as a 5-stage deterministic pipeline: 3D Structure Parsing, LLM Embedding, 3D GNN Backbone, Feature concatenation, and downstream competition-specific finalization.

## Approach

This template captures the ordered steps used by Novozymes Enzyme Stability Prediction - 1st Place (2023): prepare the data, build task-specific representations, run the main modeling or search loop, and convert outputs into the final benchmark submission.

## CDG Walkthrough

1. **3D Structure Parsing** -- This stage convert the provided PDB (Protein Data Bank) files into 3D spatial graphs, where nodes are amino acids and edges connect atoms within a specific angstrom radius (BioPython/numpy).
1. **LLM Embedding** -- This stage pass the 1D amino acid sequence through a pre-trained biological language model (ESM-2) to extract rich semantic sequence embeddings (HuggingFace).
1. **3D GNN Backbone** -- This stage train a specialized 3D Graph Neural Network (e.g., SchNet or EGNN) to process the spatial coordinates, natively handling rotational and translational equivariance (PyTorch Geometric).
1. **Feature concatenation** -- This stage combine the ESM-2 sequence embeddings and the EGNN structural embeddings into a unified vector.
1. **Regression Head** -- This stage output the continuous melting point (Tm) prediction using an MLP.

## Key Insight

The load-bearing idea is the pipeline topology: 3D Structure Parsing creates the representation needed by Regression Head, so later stages optimize the metric without solving raw-data routing from scratch.
