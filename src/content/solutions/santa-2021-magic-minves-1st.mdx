# Santa 2021 - The Magic of Magical Minves - 1st Place (2021) - Pipeline Template

## Problem

Combinatorial Optimization (String routing) solution template for Santa 2021 - The Magic of Magical Minves - 1st Place (2021), modeled as a 5-stage deterministic pipeline: Problem reduction, Distance matrix generation, Greedy initialization, LKH Optimization, and downstream competition-specific finalization.

## Approach

This template captures the ordered steps used by Santa 2021 - The Magic of Magical Minves - 1st Place (2021): prepare the data, build task-specific representations, run the main modeling or search loop, and convert outputs into the final benchmark submission.

## CDG Walkthrough

1. **Problem reduction** -- This stage translate the string permutation problem structurally into a classic Asymmetric Traveling Salesman Problem (ATSP) (Python).
1. **Distance matrix generation** -- This stage compute the exact edge weights (overlap distance) between every possible mandatory substring (numpy).
1. **Greedy initialization** -- This stage run a fast greedy search to generate a mathematically valid, but non-optimal, baseline path through the graph (NetworkX).
1. **LKH Optimization** -- This stage pass the distance matrix and baseline path into LKH-3 (a state-of-the-art heuristic solver for ATSP) to iteratively optimize the route (LKH-3 C binary).
1. **Wildcard injection** -- This stage post-process the solver's output by algorithmically injecting the allowable "wildcard" characters to squeeze out the final few characters of string length (Python).

## Key Insight

The load-bearing idea is the pipeline topology: Problem reduction creates the representation needed by Wildcard injection, so later stages optimize the metric without solving raw-data routing from scratch.
