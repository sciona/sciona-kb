# Jane Street Market Prediction - 1st Place (2021) - Pipeline Template

## Problem

Online/streaming prediction (Financial Tabular) solution template for Jane Street Market Prediction - 1st Place (2021), modeled as a 5-stage deterministic pipeline: Streaming imputation, Action space mapping, Feature dimensionality reduction, Multi-head MLP, and downstream competition-specific post-processing.

## Approach

This template captures the ordered steps used by Jane Street Market Prediction - 1st Place (2021): prepare the data, build task-specific representations, train or apply the main model, and convert outputs into the final competition submission.

## CDG Walkthrough

1. **Streaming imputation** -- This stage maintain a rolling state of previous row values to apply forward-filling for missing data dynamically (numpy).
1. **Action space mapping** -- This stage convert the single utility target into 5 distinct binary targets based on different time horizons.
1. **Feature dimensionality reduction** -- This stage train a custom autoencoder to project 130 features into a compressed space, then concatenate with original features (PyTorch).
1. **Multi-head MLP** -- This stage train a deep MLP with bottleneck layers and a custom utility-based loss function on the concatenated features (PyTorch).
1. **Streaming inference** -- This stage ingest data row-by-row via the competition API, update state arrays, and predict the binary action.

## Key Insight

The load-bearing idea is the pipeline topology: Streaming imputation creates the representation needed by Streaming inference, so later stages can optimize the competition metric without solving raw-data ingestion from scratch.
