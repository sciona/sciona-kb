1. **Robust Loss** -- This stage utilize Bi-Tempered Logistic Loss and Label Smoothing to prevent the model from confidently fitting the mislabeled dataset images.
1. **Pseudo-labeling** -- This stage generate soft labels for the 2019 dataset and merge with the 2020 dataset for retraining.
1. **Ensembling** -- This stage average predictions across 5-folds using Test Time Augmentation (TTA) (numpy).

## Key Insight
