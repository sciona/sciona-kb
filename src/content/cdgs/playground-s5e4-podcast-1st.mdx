### Automated Data Ingestion :red_circle:

### Target Definition :red_circle:

This stage map the continuous listening time target variable for regression optimization.

### AutoML Search :red_circle:

This stage execute the TabularPredictor.fit() framework utilizing the predefined "best" quality preset with a strict hardware time limit.

This stage rely entirely on the framework's internal automated feature engineering, neural network/GBDT multi-layer stacking, and out-of-fold cross-validation, bypassing explicit stage-by-stage coding.

### Prediction Extraction :red_circle:

This stage extract and format final regression outputs (pandas).
