## Key Insight

The load-bearing idea is the pipeline topology: Base Pair Probability (BPP) extraction creates the representation needed by Dual-head output, so later stages optimize the metric without solving raw-data routing from scratch.

## Pipeline Stages

### Base Pair Probability (BPP) extraction :red_circle:

This stage process RNA sequence strings through domain-specific physics simulators to generate structural probability matrices (ViennaRNA/EternaFold).

### Graph formatting :red_circle:

This stage map the RNA nucleotides to nodes and the BPP connections to edges (numpy).

### Graph Transformer Encoder :red_circle:

This stage pass the graph structure through a deep Graph Transformer, explicitly injecting the chemical structure as positional encodings (PyTorch).

### 1D CNN Encoder :red_circle:

This stage process the raw string sequence in parallel using a 1D ConvNeXT architecture to capture linear dependencies (PyTorch).

### Dual-head output :white_check_mark:

This stage fuse the representations and predict 2 continuous reactivity targets per nucleotide along the sequence using L1 loss.
