# Avito Demand Prediction - 1st Place (2018) - Pipeline Template

> This template captures the ordered steps used by Avito Demand Prediction - 1st Place (2018): prepare the data, build task-specific representations, run the main modeling or search loop, and convert outputs into the final benchmark submission.

## Key Insight

The load-bearing idea is the pipeline topology: Image feature extraction creates the representation needed by Final Blend, so later stages optimize the metric without solving raw-data routing from scratch.

## Pipeline Stages

### Image feature extraction :red_circle:

This stage pass ad images through a pre-trained VGG16/ResNet50 backbone to extract the penultimate fully-connected layer embeddings (Keras).

### Text feature extraction :red_circle:

This stage apply TF-IDF for basic word counts and FastText embeddings for semantic meaning on the ad descriptions and titles (sklearn / Gensim).

### Tabular encoding :red_circle:

This stage apply target-encoding to high-cardinality categorical variables (city, category) using multi-fold splits to prevent target leakage (pandas/category_encoders).

### Concatenation & LightGBM :red_circle:

This stage stack the image embeddings, text TF-IDF vectors, and tabular features into a massive sparse matrix and feed into LightGBM regressors.

### Multi-modal RNN architecture :red_circle:

This stage train a parallel Neural Network feeding the FastText embeddings through an RNN, concatenating with image/tabular dense layers, and predicting the target directly (PyTorch).

### Final Blend :red_circle:

This stage weight the LightGBM and Neural Network predictions (scipy).
