# ALASKA2 Image Steganalysis 1st Place — JPEG Forensics via DCT-Aware CNNs (2020)

> Detect hidden messages in JPEG photos by extracting the raw compressed data (DCT coefficients and color channels) instead of the normal image pixels, feeding this richer representation into neural networks, and averaging predictions across image rotations and multiple training runs.

{/* <ScionaViewerPlaceholder cdgId="solution.kaggle.alaska2_steganalysis_1st" readOnly={true} /> */}

## When to Use This

- Task requires detecting subtle modifications to JPEG images where the signal lives in the frequency/compression domain, not the pixel domain
- Input images are JPEG-compressed and the forensic signal is embedded in DCT coefficients (e.g., steganography, double-compression detection, tampering localization)
- Standard image classification pipelines (load-as-RGB, pretrained backbone) have plateaued because the pixel-space representation destroys the relevant signal

## Key Insight

Reading DCT coefficients directly from JPEG files (bypassing standard decompression) preserves the subtle statistical artifacts that steganography introduces in the frequency domain. Standard pixel-space loading through PIL/OpenCV applies dequantization, IDCT, chroma upsampling, and color clipping — each step destroys part of the forensic signal. The dual RGB+YCbCr 6-channel representation gives the network access to both perceptual and compression-domain features.
