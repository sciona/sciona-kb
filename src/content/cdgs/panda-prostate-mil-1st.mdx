## Pipeline Stages

### White-space culling :red_circle:

This stage run a fast thresholding heuristic across the gigapixel slide to drop 90% of the image that is empty glass/white space (OpenCV).

### N-Tile generation :red_circle:

This stage extract the top 36 most tissue-dense 256x256 patches from the remaining area (numpy).

### Grid concatenation :red_circle:

This stage stitch the 36 individual patches back together into a single 6x6 dense grid (yielding a uniform 1536x1536 image without white space) (OpenCV).

### CNN Backbone :red_circle:

This stage train an EfficientNet-B1 / SE-ResNext50 directly on the stitched grid (PyTorch).

### Ordinal Regression :red_circle:

This stage instead of classification, use an ordinal regression loss head to penalize predicting Grade 1 when the true answer is Grade 5 (PyTorch).
