## Pipeline Stages

### Object detection :white_check_mark:

This stage run YOLOv5 to detect all helmets in the video frames (Ultralytics).

**Bound atom:** `sciona.atoms.dl.architectures.yolo_object_detector`

### Temporal tracking :red_circle:

This stage link bounding boxes across frames using DeepSORT based on intersection-over-union (IoU) and CNN embeddings.

### Sensor data alignment :red_circle:

This stage project tabular player sensor data (RFID x/y coordinates) onto the 2D camera view using camera calibration matrices (OpenCV).

### Bipartite matching :red_circle:

This stage map tracked helmet clusters to projected player RFID clusters using Iterative Closest Point (ICP) and the Hungarian algorithm (scipy.optimize.linear_sum_assignment).

### Post-processing :red_circle:

This stage fix fragmented tracks and impute missing player assignments via forward-fill.
