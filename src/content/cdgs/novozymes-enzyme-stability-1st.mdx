## Pipeline Stages

### 3D Structure Parsing :red_circle:

This stage convert the provided PDB (Protein Data Bank) files into 3D spatial graphs, where nodes are amino acids and edges connect atoms within a specific angstrom radius (BioPython/numpy).

### LLM Embedding :red_circle:

This stage pass the 1D amino acid sequence through a pre-trained biological language model (ESM-2) to extract rich semantic sequence embeddings (HuggingFace).

### 3D GNN Backbone :red_circle:

This stage train a specialized 3D Graph Neural Network (e.g., SchNet or EGNN) to process the spatial coordinates, natively handling rotational and translational equivariance (PyTorch Geometric).

### Feature concatenation :red_circle:

This stage combine the ESM-2 sequence embeddings and the EGNN structural embeddings into a unified vector.

### Regression Head :red_circle:

This stage output the continuous melting point (Tm) prediction using an MLP.
