# OSIC Pulmonary Fibrosis Progression 1st Place — Multimodal Quantile Regression with CT Image Features

> Predict how a patient's lung capacity will change over time by combining features from their CT scan images with clinical measurements, training a neural network that outputs both the predicted value and how uncertain it is.

{/* <ScionaViewerPlaceholder cdgId="solution.kaggle.osic_pulmonary_1st" readOnly={true} /> */}

## When to Use This

- Regression task where the metric explicitly rewards calibrated uncertainty estimates (e.g., modified Laplace log-likelihood, CRPS)
- Multimodal data combining 3D medical images with structured tabular/temporal features
- Small patient cohort (&lt;500 patients) where end-to-end image model training would overfit but pretrained feature extraction is viable
- Prediction targets are temporal trajectories (value at future timepoints) requiring both point estimates and confidence intervals

## Key Insight

Quantile regression naturally produces calibrated confidence intervals that directly optimize the competition's modified Laplace log-likelihood metric. The spread between the 20th and 80th percentile predictions becomes the confidence estimate, meaning the model learns both where and how uncertain its predictions are. This is superior to predicting mean + variance separately because the quantile loss enforces ordering and avoids the distributional assumptions of Gaussian confidence.
