# PLAsTiCC Astronomical Classification 1st Place — Time Series Classification via GP Interpolation and Gradient Boosting

> Classify astronomical objects by extracting statistical features from their brightness measurements over time, smoothing the irregular measurements with Gaussian Processes, and combining gradient boosting with neural networks to predict object type probabilities.

{/* <ScionaViewerPlaceholder cdgId="solution.kaggle.plasticc_1st" readOnly={true} /> */}

## When to Use This

- Irregularly-sampled multi-channel time series where observations are sparse and unevenly spaced across channels
- Classification task where domain-specific feature engineering (tsfresh, cesium) is well-established for the signal type
- Severe class imbalance (>100:1 for rare classes) requiring explicit augmentation and calibrated probability outputs
- Metric is log-loss or another proper scoring rule that rewards well-calibrated probabilities over hard classifications

## Key Insight

GP interpolation converts irregularly-sampled light curves into a uniform feature space where tabular models work, while preserving uncertainty information that raw interpolation discards. The GP kernel hyperparameters (length-scale, amplitude, noise variance) themselves become discriminative features that encode the timescale and smoothness of variability -- information that summary statistics alone cannot capture.
