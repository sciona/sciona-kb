## Key Insight

## Pipeline Stages

### Bottom-up Image extraction :red_circle:

This stage process images through a Faster R-CNN object detector, strictly extracting the specific ROI (Region of Interest) pooled features for the top 36 detected objects, ignoring the background entirely (Caffe/PyTorch).

### Top-down Text embedding :red_circle:

This stage process the text question through a GRU (Gated Recurrent Unit) to generate a dense sentence vector.

### Attention alignment :red_circle:

This stage calculate a soft attention mechanism connecting the text vector to each of the 36 isolated image object vectors (PyTorch).

### Feature fusion :red_circle:

This stage utilize a Multi-modal Factorized Bilinear (MFB) pooling layer to mathematically merge the attended image features and text features (PyTorch).

### Vocabulary mapping :red_circle:

This stage output a multi-class softmax probability over the top 3,000 most common answers in the training set.
