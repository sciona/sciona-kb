# PetFinder.my Adoption Prediction 1st Place — Multimodal Ordinal Classification with Threshold Optimization

> Predict how quickly a pet will be adopted by combining information from its photos, text description, and profile data, training multiple models and then optimizing the cutoff points between adoption speed categories.

{/* <ScionaViewerPlaceholder cdgId="solution.kaggle.petfinder_adoption_1st" readOnly={true} /> */}

## When to Use This

- Ordinal classification task where the metric is Quadratic Weighted Kappa (QWK) or another ranking-sensitive ordinal metric
- Multimodal inputs (image + text + tabular) where no single modality dominates and early fusion into a flat feature matrix is feasible
- Target is ordinal with 3-7 ordered categories — few enough for threshold optimization to be tractable but too many for naive rounding to be optimal
- Competition or production metric is non-differentiable and requires post-hoc optimization rather than direct gradient-based training

## Key Insight

Nelder-Mead threshold optimization on QWK is the critical stage -- it converts continuous regression outputs into optimal ordinal bins, which is non-obvious because naive rounding loses significant QWK score. By treating the regression-to-ordinal conversion as a separate optimization problem with a derivative-free solver, the pipeline decouples learning the ranking from learning the bin boundaries, and the 4-threshold search space is small enough for Nelder-Mead to find near-global optima.

### Tabular Processing :red_circle:

### Feature Union :red_circle:

### Level-1 Models :warning:
