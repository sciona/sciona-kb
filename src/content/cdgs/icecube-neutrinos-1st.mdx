## Pipeline Stages

### Point cloud processing :red_circle:

This stage format detector hits into node features (x, y, z, time, charge) (numpy).

### Graph definition :red_circle:

This stage build a KNN graph connecting hits based strictly on spatial and temporal proximity (PyTorch Geometric).

### Graph Neural Network :red_circle:

This stage pass the graph through a DynEdge architecture (GraphNet) to allow message passing between detector nodes (PyTorch).

### Transformer block :red_circle:

This stage feed the pooled graph node embeddings into a transformer encoder for global context mapping.

### Directional loss :white_check_mark:

This stage predict 3D spherical coordinates using a von Mises-Fisher (vMF) loss function to handle angular periodicity constraints.
