# VSB Power Line Fault Detection 1st Place — Signal Processing + Hybrid Model Blend

> Detect electrical faults in power lines by cleaning up noisy voltage signals, finding discharge spikes, and combining a tree-based model on spike statistics with a neural network on the cleaned raw signal.

{/* <ScionaViewerPlaceholder cdgId="solution.kaggle.vsb_power_line_1st" readOnly={true} /> */}

## When to Use This

- High-frequency 1D signals contain transient events (spikes, discharges, anomalies) embedded in a noisy baseline
- Domain knowledge can identify meaningful signal features (peaks, frequency bands) but end-to-end models may discover additional patterns
- Signal length is large enough (>10k samples) that reshaping into a 2D matrix creates meaningful spatial structure across cycles or frequency bands
- Both interpretable feature engineering and learned representations are desired for complementary coverage

## Key Insight

Transforming 1D high-frequency signals into 2D matrices (spectrogram-like representations) allows image-based deep learning models to detect spatial patterns in the frequency-time domain that 1D models miss. The wavelet denoising is critical because corona discharge events are transient and time-localized — DWT preserves their temporal structure while FFT-based methods would smear them. The hybrid blend of handcrafted peak features with end-to-end neural features captures both known physics and unknown patterns.

### Signal to Matrix Transform :red_circle:
