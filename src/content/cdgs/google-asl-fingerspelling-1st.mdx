## Key Insight

The load-bearing idea is the pipeline topology: Landmark interpolation creates the representation needed by CTC Loss Translation, so later stages optimize the metric without solving raw-data routing from scratch.

## Pipeline Stages

### Landmark interpolation :red_circle:

This stage fill dropped MediaPipe 3D coordinate frames using linear interpolation to ensure a continuous time series (pandas).

### Affine Augmentation :red_circle:

This stage apply random 3D rotations, scaling, and sheer to the skeletal coordinates to simulate different camera angles during training (numpy).

### Feature engineering :red_circle:

This stage calculate Euclidean distance, velocity, and angles between specific finger joints (thumb to index, wrist to pinky) explicitly (numpy).

### CNN + Transformer Architecture :red_circle:

This stage pass the coordinates through a 1D CNN to extract local temporal features, followed by a Transformer Encoder for global sequence context (PyTorch).

### CTC Loss Translation :red_circle:

This stage decode the sequence into raw text tokens using Connectionist Temporal Classification (CTC) loss to handle the misalignment between sequence length and text length (PyTorch).
