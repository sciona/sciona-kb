## Key Insight

## Pipeline Stages

### Keypoint parsing :red_circle:

This stage load Google MediaPipe spatial coordinates tracking hands, face, and pose frame-by-frame (numpy).

### Spatial normalization :red_circle:

This stage translate all keypoints so the chest center is (0,0,0) and scale hand sizes uniformly to neutralize distance-to-camera variance (numpy).

### Transformer Encoder :red_circle:

This stage pass the 3D coordinate sequence directly into a Transformer Encoder block to capture spatial/temporal features without any CNN architectures (PyTorch).

### Text Decoder :red_circle:

This stage pass the encoder embeddings to an autoregressive Transformer Decoder trained via Cross-Entropy to output text tokens (PyTorch).

### Beam Search :red_circle:

This stage execute Beam Search during inference to select the most probable text string, utilizing length penalties to prevent prematurely short translations (Python).
