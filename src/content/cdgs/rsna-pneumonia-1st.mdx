# RSNA Pneumonia Detection 1st Place — Dual-Detector Box Ensemble

> Find pneumonia in chest X-rays by running two different object detection models (RetinaNet and Mask R-CNN) on the same images, removing duplicate detections within each model, then merging their bounding boxes based on how much they overlap.

{/* <ScionaViewerPlaceholder cdgId="solution.kaggle.rsna_pneumonia_1st" readOnly={true} /> */}

## When to Use This

- 2D medical image detection where targets are diffuse opacities with poorly defined boundaries, benefiting from detector diversity
- Single-class or few-class detection task where two complementary architectures (single-stage + two-stage) can be trained with available compute
- DICOM inputs requiring careful windowing and conservative augmentation to preserve diagnostic content

## Key Insight

Single-stage detectors (RetinaNet) and two-stage detectors (Mask R-CNN) make systematically different errors: RetinaNet is faster but produces more false positives on ambiguous regions, while Mask R-CNN is more precise but may miss subtle findings. The IoU-based box ensemble exploits this complementarity — detections agreed upon by both models receive boosted confidence, while single-model detections are penalized, effectively using inter-model agreement as a proxy for detection reliability.

## Pipeline Stages

### DICOM Extraction :red_circle:

Convert the raw medical image files from their specialized hospital format into standard image files that the detection models can process.

### X-ray Augmentation :warning:

Create variations of each X-ray by slightly rotating, zooming, and adjusting brightness so the models learn to handle natural variation in how X-rays are taken.

**Bound atom:** `sciona.atoms.dl.image_augmentation.affine_transform_centered`

### RetinaNet Detector :warning:

Train a fast object detection model using a special loss function that focuses on hard-to-detect cases, producing bounding boxes around suspected pneumonia regions.

**Bound atom:** `sciona.atoms.dl.architectures.resnet_family_backbone`

### Mask R-CNN Detector :warning:

Train a second, more thorough object detection model that first proposes candidate regions then classifies them, providing a complementary set of detections.

**Bound atom:** `sciona.atoms.dl.architectures.resnet_family_backbone`

### NMS Postprocessing :white_check_mark:

When a model detects the same pneumonia region multiple times with slightly different boxes, keep only the most confident one and discard the rest.

**Bound atom:** `sciona.atoms.dl.detection.nms`

### Box Ensemble :warning:

Combine the detections from both models: if both models found a pneumonia region in the same spot, merge them into one high-confidence box; if only one model found it, keep it at lower confidence.

**Bound atom:** `sciona.atoms.dl.detection.wbf`
