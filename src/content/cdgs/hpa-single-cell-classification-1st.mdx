## Key Insight

The load-bearing idea is the pipeline topology: Instance Segmentation creates the representation needed by Label assignment, so later stages optimize the metric without solving raw-data routing from scratch.

## Pipeline Stages

### Instance Segmentation :red_circle:

This stage utilize the pre-trained Cellpose algorithm on the 4-channel imagery (microtubules, nucleus, etc.) to generate precise masks for every individual cell in the image (Cellpose).

### Crop Extraction :red_circle:

This stage extract thousands of individual cell image crops based on the generated masks (numpy).

### Image-Level Training :red_circle:

This stage train an EfficientNet model on the full, uncropped images utilizing the image-level ground truth labels (since cell-level labels do not exist) (PyTorch).

### Multi-Instance Learning (MIL) :red_circle:

This stage extract the cell-level crops, pass them through the trained network, and use a specialized MIL pooling layer (e.g., max-pooling across cells) to map cell-level probabilities back to the image-level label.

### Label assignment :red_circle:

This stage assign the highest probability classes back to the individual cell instances for final scoring.
