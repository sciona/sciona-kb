## Key Insight

## Pipeline Stages

### Audio feature extraction :white_check_mark:

This stage generate standard Mel Spectrograms from continuous audio files (librosa).

**Bound atom:** `sciona.atoms.signal.audio_speech.log_mel_spectrogram`

### Teacher Model Training :red_circle:

This stage train a massive, high-accuracy ResNeSt-50 model on the spectrograms without any hardware constraints (PyTorch).

### Student Model Training (Distillation) :red_circle:

This stage train a micro MobileNetV3 architecture using the soft probability labels output by the Teacher model rather than the hard ground-truth labels (PyTorch).

### TFLite Conversion :red_circle:

This stage convert the trained PyTorch MobileNetV3 into an ONNX format, then strictly quantize it to INT8 precision using TensorFlow Lite to ensure sub-50ms inference times (ONNX/TFLite).

### On-device Thresholding :red_circle:

This stage implement fixed threshold bounds inside the TFLite wrapper to filter predictions before final return.
