# Otto Group Product Classification - 1st Place (2015) - Pipeline Template

> This template captures the ordered steps used by Otto Group Product Classification - 1st Place (2015): prepare the data, build task-specific representations, train or apply the main model, and convert outputs into the final competition submission.

## When to Use This

- The new task has the same broad problem type: Multi-label classification.
- The available data supports the same ordered transformations and intermediate artifacts as the documented winning pipeline.
- Offline batch processing is acceptable, including feature extraction, model ensembling, or post-processing stages.

## Key Insight

The load-bearing idea is the pipeline topology: Numerical Engineering creates the representation needed by Final Ensemble, so later stages can optimize the competition metric without solving raw-data ingestion from scratch.

## Pipeline Stages

### Numerical Engineering :red_circle:

This stage scale the base data and generate Gaussian Mixture Model (GMM) clustering assignments as entirely new synthetic features (sklearn.mixture).

### Interaction Features :red_circle:

This stage create hard-coded ratios, mathematical products, and statistical measures combining the original categorical and numerical columns (pandas).

### Stratification :red_circle:

This stage execute a Repeated Multilabel Stratified K-Fold cross-validation strategy specifically designed to perfectly preserve class distributions across all folds (iterative-stratification).

### Base Modeling :red_circle:

This stage train XGBoost and LightGBM models wrapped inside a MultiOutputClassifier to handle the diverse label space (sklearn.multioutput, xgboost, lightgbm).

### Level-2 Meta-Modeling :red_circle:

This stage extract the Out-of-Fold probabilities from the GBDTs and use them to train secondary Neural Networks and AdaBoost models (torch, sklearn.ensemble).

### Final Ensemble :red_circle:

This stage blend the Level-2 meta-predictions utilizing a carefully weighted geometric and arithmetic mean.
