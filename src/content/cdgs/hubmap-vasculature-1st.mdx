# HuBMAP Hacking the Human Vasculature - 1st Place (2023) - Pipeline Template

> This template captures the ordered steps used by HuBMAP Hacking the Human Vasculature - 1st Place (2023): prepare the data, build task-specific representations, train or apply the main model, and convert outputs into the final competition submission.

## When to Use This

- The new task has the same broad problem type: Semantic/instance segmentation.
- The available data supports the same ordered transformations and intermediate artifacts as the documented winning pipeline.
- Offline batch processing is acceptable, including feature extraction, model ensembling, or post-processing stages.

## Key Insight

The load-bearing idea is the pipeline topology: Data Concatenation creates the representation needed by Post-Processing Dilation, so later stages can optimize the competition metric without solving raw-data ingestion from scratch.

## Pipeline Stages

### Data Concatenation :red_circle:

This stage merge multiple disparate origin datasets (dataset1 and dataset2) to maximize sample size and force the model to learn invariant representations (pandas).

### Model Initialization :red_circle:

This stage initialize an RTMDet-x baseline architecture tailored for instance segmentation over bounding box prediction (mmdetection).

### Weight Averaging :red_circle:

This stage apply Exponential Moving Average (EMA) to the model weights continuously during training. This technique acts as a regularizer and heavily stabilizes convergence in noisy medical imagery (torch.optim.swa_utils).

### Mask Prediction :warning:

This stage generate baseline mask predictions using the standard segmentation head operating on the spatial features.

**Bound atom:** `sciona.atoms.dl.architectures.unet_2d_segmentation`

### Post-Processing Dilation :red_circle:

This stage apply morphological dilation to the predicted binary masks to uniformly expand boundaries. This purely algorithmic stage significantly boosts the evaluation metric without requiring further network training (scipy.ndimage or cv2).
