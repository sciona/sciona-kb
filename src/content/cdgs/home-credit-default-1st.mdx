## Key Insight

## Pipeline Stages

### Relational aggregation :red_circle:

This stage write custom SQL-style groupby statements to aggregate historical credit bureau, previous application, and installment tables to the user level (pandas).

### Time-decay weighting :red_circle:

This stage apply heavier weights to recent financial actions during aggregation (e.g., recent late payments penalize more than 5-year-old late payments) (pandas).

### Feature selection :red_circle:

This stage train a massive LightGBM model with 2000+ features, computing SHAP values, and dropping the bottom 50% least important features (LightGBM/SHAP).

### Null imputation :red_circle:

This stage replace NaN values with heavily negative out-of-bounds numbers to let tree models naturally group missing values (pandas).

### Ensembling :red_circle:

This stage blend Ridge Regression, LightGBM, XGBoost, and Neural Network probabilities using rank-averaging.
