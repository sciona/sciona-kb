# APTOS 2019 Blindness Detection - 1st Place (2019) - Pipeline Template

- The new task has the same broad problem type: Ordinal Classification (Medical Imaging).

## Key Insight

The load-bearing idea is the pipeline topology: Image standardization creates the representation needed by Threshold Calibration, so later stages optimize the metric without solving raw-data routing from scratch.

## Pipeline Stages

### Image standardization :red_circle:

This stage apply Ben Graham's preprocessing method: crop to the circular retina, subtract local median color to highlight blood vessels, and resize (OpenCV).

### CNN Backbone :red_circle:

This stage train EfficientNet and SE-ResNeXt architectures initialized with ImageNet weights (PyTorch).

### Quadratic Weighted Kappa (QWK) Loss :red_circle:

This stage optimize the network using a smooth QWK loss function, heavily penalizing off-by-2 errors (e.g., predicting 0 when truth is 4) more than off-by-1 errors (custom PyTorch loss).

### Pre-training & Fine-tuning :red_circle:

This stage pre-train the model on massive public datasets from past 2015 competitions, then strictly fine-tune only the last 2 layers on the smaller 2019 competition dataset.

### Threshold Calibration :red_circle:

This stage use Nelder-Mead optimization to perfectly tune the decision boundaries between classes [0, 1, 2, 3, 4] to maximize the QWK metric.
