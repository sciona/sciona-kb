## Pipeline Stages

### Temporal unrolling :white_check_mark:

This stage melt the wide historical sales matrix into a long, row-based format suitable for standard machine learning (pandas).

**Bound atom:** `sciona.atoms.dl.video_temporal.temporal_unroll`

### Recursive feature engineering :red_circle:

This stage calculate rolling means, standard deviations, and lag features at 7, 14, and 28-day intervals (pandas).

### Objective function optimization :red_circle:

This stage configure a LightGBM model utilizing a Tweedie objective function, perfectly suited for data with a massive number of exact zero values (LightGBM).

### Magic Multiplier :red_circle:

This stage mathematically shift all predictions upward by a strict constant multiplier (e.g., 1.03) to manually account for the competition's anomalous "empty" days (numpy).

### Hierarchical Reconciliation :red_circle:

This stage aggregate the store-level/item-level predictions bottom-up to ensure they perfectly match the required state-level and national-level hierarchical constraints (scipy).
