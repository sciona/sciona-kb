## Pipeline Stages

### Feature Augmentation :red_circle:

This stage recompute all non-deterministic search tree features 10 distinct times and sample directly from non-held-out runs. This topological stage intentionally injects structural noise into the training rows to combat overfitting (pandas, numpy).

### Target Smoothing :red_circle:

This stage recompute the competition's target labels 5 times to estimate a significantly cleaner, lower-variance "real" target value.

### Quantile Regression Training (Lower) :white_check_mark:

This stage train a regressor explicitly targeted toward the negative (-1) direction utilizing a strict quantile loss function (alpha=0.2) (lightgbm or xgboost).

**Bound atom:** `sciona.atoms.dl.time_series.pinball_loss`

### Quantile Regression Training (Upper) :white_check_mark:

This stage train an independent regressor targeted toward the positive (+1) direction using an inverse quantile loss (alpha=0.8).

**Bound atom:** `sciona.atoms.dl.time_series.pinball_loss`

### Directional Ensembling :red_circle:

This stage blend the initial negative predictions heavily with the alpha=0.2 model and positive predictions with the alpha=0.8 model using a fixed.
