## Key Insight

## Pipeline Stages

### Target normalization :warning:

This stage adjust the raw target (future return) by subtracting the market-wide index return to isolate the alpha of individual stocks (pandas).

**Bound atom:** `sciona.atoms.fintech.signal_processing.alpha_isolation`

### Technical features :red_circle:

This stage calculate standard financial indicators (moving averages, volatility, RSI) over multiple windows (pandas/TA-Lib).

### Factor modeling :red_circle:

This stage train LightGBM regressors to predict the continuous adjusted return.

### Rank transformation :white_check_mark:

This stage ignore the absolute values of the predictions, immediately converting the regression outputs into a strictly ranked integer list [1 to 2000] for all stocks on a given day (scipy.stats.rankdata).

**Bound atom:** `sciona.atoms.ml.tabular.gradient_boosting.rank_transform`

### Sharpe Ratio smoothing :red_circle:

This stage apply an Exponential Moving Average (EMA) to the daily predictions over time to prevent the model from rapidly flipping ranks, which directly maximizes the Sharpe Ratio evaluation metric.
