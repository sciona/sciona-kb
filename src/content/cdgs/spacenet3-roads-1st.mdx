## Key Insight

The load-bearing idea is the pipeline topology: Multispectral stacking creates the representation needed by Path routing, so later stages optimize the metric without solving raw-data routing from scratch.

## Pipeline Stages

### Multispectral stacking :red_circle:

This stage stack panchromatic and multi-spectral GeoTIFF bands into a single 8-channel image tensor (rasterio).

### Multi-class segmentation :red_circle:

This stage train a deep U-Net (ResNet34 backbone) to predict 3 distinct spatial masks: roads, intersections, and background (PyTorch).

### Skeletonization :red_circle:

This stage pass the binary road mask through a morphological skeletonization algorithm to reduce roads to 1-pixel wide continuous lines (skimage.morphology).

### Graph generation :white_check_mark:

This stage trace the skeletonized pixels to define nodes (intersections) and edges (road segments) (NetworkX).

**Bound atom:** `sciona.atoms.dl.skeletonization.skeleton_to_graph`

### Path routing :red_circle:

This stage implement an A* search heuristic to bridge small pixel gaps in the graph caused by tree cover or cloud shadows (Python).
