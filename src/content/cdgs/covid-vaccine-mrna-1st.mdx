# Stanford COVID Vaccine mRNA 1st Place — Per-Nucleotide Degradation Prediction

> Predict how fast each position in an RNA strand degrades by reading the sequence left-to-right and right-to-left with one model, while a second model reasons about which bases are physically paired, then merging both views before making the final prediction.

{/* <ScionaViewerPlaceholder cdgId="solution.kaggle.covid_vaccine_mrna_1st" readOnly={true} /> */}

## When to Use This

- Per-position prediction on biological sequences where both linear order and structural (base-pairing or contact) topology carry information
- Secondary structure or contact maps are available or can be predicted, providing a meaningful graph for GCN message passing
- Output is a dense per-residue regression or classification (not a single sequence-level label)

## Key Insight

RNA degradation is governed by both sequential context (neighboring nucleotides) and structural context (physically paired bases that may be distant in sequence). A single-modality model captures only one view. The dual-branch fusion forces complementary learning: the RNN captures sequential motifs while the GCN propagates information along base-pair edges, and their concatenation at every position gives the output head both views simultaneously.

## Pipeline Stages

### Secondary Structure Parsing :red_circle:

Use physics-based RNA folding software to predict which bases in the strand pair up, and store those pairings as a connectivity matrix.

### Node Feature Generation :white_check_mark:

**Bound atom:** `sciona.atoms.graph_features.node_onehot_positional`

### Sequence-to-Sequence Model :white_check_mark:

Read the RNA sequence forwards and backwards with a recurrent neural network to build a context-aware representation at each position.

**Bound atom:** `sciona.atoms.dl.architectures.recurrent_sequence_model`

### Graph Neural Network :white_check_mark:

Use a graph neural network to learn representations based on which bases are physically paired together in the RNA's 3D structure.

**Bound atom:** `sciona.atoms.dl.architectures.graph_neural_network`

### Feature Fusion :red_circle:

Merge the sequence-reading model's output with the structure-reading model's output at each RNA position.

### Output Head :white_check_mark:

A simple linear layer that converts the merged representation into the final degradation rate predictions for each position.

**Bound atom:** `sciona.atoms.dl.architectures.linear_prediction_head`
