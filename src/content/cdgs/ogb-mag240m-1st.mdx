## Key Insight

## Pipeline Stages

### Node text extraction :red_circle:

This stage run RoBERTa across all 240 million paper nodes to generate dense 768-dimensional text embeddings (HuggingFace).

### Graph sampling :white_check_mark:

This stage construct a heterogeneous neighborhood sampling pipeline to handle the massive graph size without encountering CPU out-of-memory errors (DGL).

**Bound atom:** `sciona.atoms.dl.architectures.heterogeneous_graph_sampler`

### Message passing :red_circle:

This stage train a multi-layer GraphSAGE architecture, passing the RoBERTa embeddings iteratively through the citation edges (PyTorch / DGL).

### Label Propagation :red_circle:

This stage execute a classic Label Propagation Algorithm (LPA) over the graph's known labels to generate a structural baseline score (NetworkX).

### Blending :red_circle:

This stage dynamically combine the deep GraphSAGE predictions with the structural LPA predictions using a learned weight array (numpy).
