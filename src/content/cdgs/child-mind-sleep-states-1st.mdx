## Pipeline Stages

### Time-series unrolling :red_circle:

This stage load millions of continuous accelerometer readings (angle, magnitude) efficiently via Parquet (polars).

### Rolling statistics :red_circle:

This stage compute rolling mean, variance, and difference across 1-minute, 5-minute, and 30-minute intervals (polars).

### 1D U-Net Segmentation :red_circle:

This stage treat the sequence exactly like a 1D image; pass it through a 1D U-Net backbone to output a probability per minute of "awake" vs "asleep" (PyTorch).

### Transformer Smoothing :red_circle:

This stage pass the U-Net embeddings through a Transformer layer to capture long-range daily circadian rhythms (PyTorch).

### Peak finding :red_circle:

This stage run SciPy's find_peaks on the derivative of the probability output to locate the exact discrete minute the user transitioned states (scipy.signal).
