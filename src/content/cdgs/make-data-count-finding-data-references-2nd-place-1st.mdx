## Key Insight

## Pipeline Stages

### External Data Gathering :red_circle:

This stage execute automated scripts to download supplementary open-access corpus data directly from PubMed. This step physically augments the heavily constrained training set prior to any modeling (requests, beautifulsoup).

### RegEx Filtering :warning:

This stage apply a rigid, regular expression-based programmatic system to detect common dataset name variations and filter out severe noise strings (re).

**Bound atom:** `sciona.atoms.ml.text_nlp.clean_text`

### Text Classification Model :red_circle:

This stage train a robust transformer-based text classifier on the augmented dataset, entirely bypassing structurally unstable NER models that failed to converge on the small dataset (transformers).

### Cross-Validation :red_circle:

This stage train the models across multiple strict folds to stabilize severe early-epoch convergence issues explicitly caused by the microscopic dataset size.

### Ensembling :red_circle:

This stage average the output classification probabilities across all stable folds.
