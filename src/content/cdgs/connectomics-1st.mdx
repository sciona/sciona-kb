    description: Apply a causal/acausal weighted moving average using np.roll along the time axis. Multiple filter kernels (symmetric 3-point, causal 4-point, asymmetric) are applied independently and ensembled later.
    dejargonized_description: Smooth the raw fluorescence signal to suppress single-frame noise, using several different smoothing windows that will be combined later.

    guarantees:
      - Output shape matches input. Temporal alignment preserved (np.roll is circular).
    family_notes:
      - Uses np.roll for circular shift to avoid edge artifacts in correlation computation.
      - The calcium indicator itself is a low-pass filter of spike trains; this additional smoothing suppresses single-frame noise without full deconvolution.
    matched_primitive: ""

  - stage_id: fluorescence_hard_threshold
    name: Fluorescence Hard Threshold
    description: Zero all values below a threshold, keeping values above unchanged. Applied after differencing, so it operates on the derivative — zeroing sub-threshold noise to focus on co-activation events.
    dejargonized_description: Discard small changes and keep only the sharp upward spikes that likely represent actual neural firing events.
    concept_type: signal_filter
    inputs:

        required: true
    preconditions:
      - Derivative signal is available.
    guarantees:
      - Output is non-negative. Zero where input was below threshold.
    family_notes:
      - This is a sparse activation detector. The threshold is NOT selected by cross-validation — it is swept densely and all results are ensembled (see threshold_sweep_ensemble stage).
    matched_primitive: ""
  - stage_id: global_activity_sample_reweighting
    name: Global Activity Sample Reweighting
    description: "Nonlinear per-timestep reweighting that suppresses samples during global burst periods and amplifies moderate-activity windows. For each timestep i, computes Sum4[i] = total activity across all neurons, then transforms: X[i,:] = (X[i,:] + 1)^(1 + 1/Sum4[i]). The exponent is further raised to a power (1.4, 1.6, or 1.9) depending on the regime of relative activity."
    dejargonized_description: When many neurons fire at once (a network-wide burst), reduce the influence of those time steps because the co-firing is due to the burst, not direct connections. Amplify time steps where only a few neurons are active.
    concept_type: signal_transform
    inputs:
      - name: thresholded_signal
        type_desc: NDArray[np.float64]
        constraints: Shape (T, N). Sparse activation matrix.
        required: true
    outputs:
      - name: reweighted_signal
        type_desc: NDArray[np.float64]
        constraints: Shape (T, N). Burst-deconfounded activation matrix.
        required: true
    preconditions:
      - Input is non-negative (post-threshold).
    guarantees:
      - Output preserves relative structure within each timestep. Global burst periods are suppressed.
    family_notes:
      - KEY WINNING TECHNIQUE. The (x+1)^(1+1/S) form with regime-dependent outer exponents is novel to this solution.
      - Credited to Stetter et al. (2012) for the general idea, but the specific nonlinear form is original.
      - Generalizes beyond calcium imaging to any 'infer graph from multivariate time series' problem where global events create spurious correlations.
    matched_primitive: ""
  - stage_id: pca_precision_matrix
    name: PCA Precision Matrix
    description: Fit sklearn PCA(whiten=True, n_components=0.8*N) on the preprocessed data, then call pca.get_precision() to obtain the precision matrix (inverse covariance), and negate it. Uses PCA with rank reduction to 80% of components as implicit spectral regularization instead of L1 penalization (Graphical Lasso).
    dejargonized_description: Compute which neurons are partially correlated (connected) by inverting the covariance matrix, using PCA to throw away the noisiest 20% of variation first so the inversion is stable.
    concept_type: dimensionality_reduction

        constraints: Shape (N, N). Negated precision matrix (partial correlations). Symmetric.

    preconditions:

    guarantees:
      - Output is symmetric (N, N). Diagonal removed downstream.
    family_notes:
      - Retaining 80% of components filters out the noise subspace, acting as spectral shrinkage. Avoids the O(N^3) cost of Graphical Lasso on large matrices.

    name: Threshold Sweep Ensemble
    description: Iterate over 120 threshold values × 2-4 filter types. For each combination, run the full preprocessing + PCA precision pipeline. Aggregate as weighted sum. This is hyperparameter-free Bayesian model averaging — rather than selecting a single best threshold, average over the dense grid.
    dejargonized_description: Instead of guessing the best sensitivity setting, try 120 different settings and average all the results, which reliably outperforms picking any single one.

        required: true
    preconditions:
      - All inner pipeline stages are available.
    guarantees:
      - Output is a smooth, hyperparameter-free connectivity estimate.
    family_notes:
      - Meta-strategy, not a standalone primitive. Wraps the inner pipeline (filter → diff → threshold → reweight → PCA → normalize) in an outer loop.

    matched_primitive: ""
  - stage_id: temporal_precedence_directivity
    name: Temporal Precedence Directivity
    description: For each ordered pair (i, j), count timesteps where x_j[t+1] falls in [x_i[t]+0.2, x_i[t]+0.5]. Build asymmetric count matrix, compute count - count^T, scale to [0,1]. Combined at 0.3% weight as a directional tiebreaker.
    dejargonized_description: "Detect which direction connections go by looking for a specific pattern: if neuron i fires at time t, neuron j should show a specific-sized increase at time t+1 if i connects to j."

        constraints: Shape (N, N). Asymmetric directivity score in [0, 1].
        required: true
    preconditions:

    guarantees:
      - Output is asymmetric (captures directionality). Scale [0, 1].
    family_notes:
      - The [+0.2, +0.5] interval is tuned to calcium indicator biophysics.

    matched_primitive: ""

# Connectomics 1st Place — Graph Inference from Calcium Fluorescence

> Figure out which neurons are connected by looking at when they fire together, after removing spurious correlations caused by whole-brain activity bursts, and averaging over many sensitivity settings instead of picking one.

{/* <ScionaViewerPlaceholder cdgId="solution.kaggle.connectomics_1st" readOnly={true} /> */}

## When to Use This

- Inferring pairwise graph structure from multivariate time series where nodes fire/activate simultaneously
- Global confounding events (bursts, market shocks, weather fronts) create spurious pairwise correlations that must be suppressed
- No labeled edges available — fully unsupervised graph inference from temporal co-activation patterns
- Threshold hyperparameter is unknown and cross-validation is impractical (no ground-truth edges for tuning)

## Key Insight

Burst deconfounding via nonlinear reweighting (x+1)^(1+1/S) suppresses global activity artifacts before precision matrix estimation. Dense threshold ensembling integrates out the threshold hyperparameter, equivalent to Bayesian model averaging under a uniform prior.
