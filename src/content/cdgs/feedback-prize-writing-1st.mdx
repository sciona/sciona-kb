# Feedback Prize — Evaluating Student Writing 1st Place — NLP Span Detection via DeBERTa Token Tagging

> Break long essays into overlapping chunks, train a model to label each word as the beginning, inside, or outside of a discourse element, then merge overlapping predictions from multiple models and discard unreasonably short spans.

{/* <ScionaViewerPlaceholder cdgId="solution.kaggle.feedback_prize_writing_1st" readOnly={true} /> */}

## When to Use This

- Task requires extracting contiguous text spans with class labels from long documents (>512 tokens)
- Annotations are at character-level offsets and must be mapped to subword token boundaries
- Multiple overlapping span predictions need to be merged into consensus spans (multi-model ensembling)
- Discourse elements or entity types have minimum length priors that can be exploited for filtering

## Key Insight

Text-adapted Weighted Box Fusion is the critical ensembling innovation: by treating character-offset spans as 1D bounding boxes and computing IoU as character overlap ratio, it merges multi-seed span predictions into high-confidence consensus spans. This is far more effective than majority voting on token labels because it resolves boundary disagreements smoothly rather than discretely.

## Pipeline Stages

### Sliding Window Tokenization :warning:

Break long essays into overlapping pieces small enough for the model to process, using the model's own text splitter.

**Bound atom:** `sciona.atoms.dl.video_temporal.sliding_windows`

### Character to Token Offset Mapping :warning:

Convert the human annotations (which mark character positions) into per-token labels that the model can learn from.

**Bound atom:** `sciona.atoms.ml.text_nlp.char_to_token_offsets`

### Sequence Tagging Training :white_check_mark:

Fine-tune a large language model to predict for each word whether it begins, continues, or is outside a discourse element.

**Bound atom:** `sciona.atoms.dl.architectures.sequence_tagging_trainer`

### Span Extraction :warning:

Convert the model's per-word predictions back into contiguous text spans with start and end positions in the original essay.

**Bound atom:** `sciona.atoms.ml.text_nlp.bio_decode`

### Text Weighted Box Fusion :white_check_mark:

Combine predictions from multiple models by merging overlapping spans into a single best span, weighted by each model's confidence.

**Bound atom:** `sciona.atoms.dl.detection.wbf_1d`

### Length Constraint Filtering :red_circle:

Remove predictions that are unreasonably short for their type, since real discourse elements like thesis statements or evidence spans require a minimum number of words.
