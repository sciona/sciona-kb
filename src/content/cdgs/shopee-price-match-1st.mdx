# Shopee Price Match Guarantee 1st Place — Multimodal ArcFace Retrieval with Graph Matching

> Find matching products by computing visual and textual similarity scores, combining them, finding all pairs above a confidence threshold, and grouping connected matches together.

{/* <ScionaViewerPlaceholder cdgId="solution.kaggle.shopee_price_match_1st" readOnly={true} /> */}

## When to Use This

- Multimodal matching/retrieval where both image and text signals are available and complementary
- Near-duplicate or same-entity detection at scale (>10K items) requiring pairwise similarity
- Fine-grained visual similarity where class boundaries are tight (e.g., same product vs. similar product)
- Multilingual text where semantic matching must work across languages

## Key Insight

ArcFace margin loss transforms the embedding space so that cosine similarity becomes a reliable proxy for semantic equivalence: it enforces an angular margin between classes, meaning the threshold for 'same product' is geometrically well-separated from 'different product'. This makes the downstream thresholding step robust rather than fragile. Connected components then propagate transitive matches, recovering groups that no single pairwise comparison would capture.

## Pipeline Stages

### Image Embedding :warning:

Convert each product image into a compact 512-number fingerprint using a deep neural network trained so that images of the same product end up close together and different products end up far apart.

### Embedding Fusion :warning:

Combine the image and text fingerprints into one unified fingerprint and normalize its length.

**Bound atom:** `sciona.atoms.dl.embeddings.l2_normalize`

### Pairwise Similarity :white_check_mark:

Compare every product's fingerprint against every other product's fingerprint to find the most similar pairs, using GPU acceleration for speed.

**Bound atom:** `sciona.atoms.dl.embeddings.cosine_similarity_matrix`

### Dynamic Thresholding :red_circle:

Decide which product pairs are actual matches by applying a confidence cutoff, potentially adjusting the cutoff based on how similar each product's neighbors are.

### Connected Components :white_check_mark:

Group all matched product pairs into clusters by following chains of matches: if A matches B and B matches C, then A, B, and C are all in the same group.

**Bound atom:** `sciona.atoms.algorithmic.graph.utilities.connected_components`
