## Pipeline Stages

### Log1p Transformation :red_circle:

This stage strictly apply log1p to the Wikipedia page views to suppress massive traffic spikes and stabilize neural network gradients (numpy).

### Missing value masking :red_circle:

This stage tag NaN values with an explicit boolean flag and fill with 0 to inform the network of missing data mechanically (pandas).

### Seq2Seq Architecture :red_circle:

This stage construct an LSTM Encoder-Decoder network. The encoder ingests the trailing 60 days of traffic, passing the hidden state to the decoder which predicts the next 60 days autoregressively (TensorFlow).

### Exogenous concatenation :red_circle:

This stage append Wikipedia page metadata (language, domain) to the LSTM inputs at every timestep.

### SMAPE optimization :red_circle:

This stage train the entire network natively on a smooth approximation of the SMAPE (Symmetric Mean Absolute Percentage Error) loss function (TensorFlow).
