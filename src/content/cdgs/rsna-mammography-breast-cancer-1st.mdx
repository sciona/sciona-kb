## Pipeline Stages

### DICOM preprocessing :red_circle:

This stage extract 16-bit medical images using strict VOI LUT (Value of Interest Look-Up Table) transformations to ensure correct tissue contrast, cropping heavily to remove black background (pydicom/cv2).

### ROI extraction :red_circle:

This stage train a lightweight YOLO model to predict exactly where the breast tissue is, drastically cropping the image to discard empty space (Ultralytics).

### Multi-view grouping :red_circle:

This stage strictly group the CC (top-down) and MLO (side-angle) views of the same breast into a single batched input (pandas).

### Multi-view CNN :red_circle:

This stage pass both views simultaneously through an EfficientNet backbone (PyTorch).

### Probabilistic pooling :red_circle:

This stage pool the embeddings of both views utilizing an Attention mechanism before passing them through the final classification head to output the cancer probability.
