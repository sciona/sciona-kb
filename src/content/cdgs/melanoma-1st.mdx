# SIIM-ISIC Melanoma Classification 1st Place — Multi-Modal CNN with Pseudo-Labeling and Rank Averaging

> Detect skin cancer from photographs by training image-recognition networks with realistic data distortions, combining image features with patient information, adding pseudo-labeled historical data, and averaging rankings across many models.

{/* <ScionaViewerPlaceholder cdgId="solution.kaggle.melanoma_1st" readOnly={true} /> */}

## When to Use This

- Medical image classification with extreme class imbalance (&lt;5% positive rate) where domain-specific augmentations can simulate real-world artifacts
- Patient metadata (age, sex, anatomical site) provides strong clinical priors that complement visual features
- Historical or external unlabeled datasets from the same domain are available for pseudo-labeling to expand the small labeled training set
- Multiple backbone architectures and resolutions are feasible and rank-average ensembling can reduce variance

## Key Insight

Domain-specific augmentations (microscope borders, hair overlay, CutMix) are the primary regularizer for extreme class imbalance — they force the model to learn lesion-intrinsic features rather than memorizing imaging artifacts. Metadata concatenation at the penultimate layer provides clinical priors (e.g., melanoma is more common at certain ages and anatomical sites) that pure image models cannot learn from pixels alone. Pseudo-labeling on historical competition data expands the effective positive sample count, which is the binding constraint at &lt;2% prevalence.

## Pipeline Stages

### Microscope Augmentation :warning:

Add realistic distortions to training images like circular borders, fake hairs, and image mixing to help the model handle real-world dermoscopic variation.

**Bound atom:** `sciona.atoms.dl.image_augmentation.cutmix_apply`

### Image Backbone Training :warning:

Train several powerful image-recognition networks on the skin images, using a special loss function that pays extra attention to the rare cancer cases.

**Bound atom:** `sciona.atoms.dl.architectures.efficientnet_backbone`

### Metadata Concatenation :warning:

Combine the image features extracted by the neural network with patient information like age, sex, and where the lesion is on the body.

**Bound atom:** `sciona.atoms.dl.tabular.entity_embedding_lookup`

### Meta Head :warning:

Feed the combined image and patient features through a small neural network to produce the final cancer probability.

**Bound atom:** `sciona.atoms.dl.architectures.linear_prediction_head`

### Pseudo-Labeling :white_check_mark:

Apply trained models to older competition datasets to generate predicted labels, then add these extra labeled examples to the training data.

**Bound atom:** `sciona.atoms.ml.tabular.gradient_boosting.extract_pseudo_labels`

### Rank Average Ensemble :white_check_mark:

Combine predictions from many different models by converting each to rankings, averaging the rankings, to get a final robust prediction.

**Bound atom:** `sciona.atoms.dl.image_augmentation.fold_ensemble_average`
