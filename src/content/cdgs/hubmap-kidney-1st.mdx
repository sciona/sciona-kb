# HuBMAP Hacking the Kidney 1st Place — Gigapixel Glomeruli Segmentation (2021)

> Segment kidney structures (glomeruli) from enormous microscope images by cutting them into overlapping tiles, normalizing the staining colors, training a segmentation neural network with a specialized loss function, and stitching the predictions back together into a full-resolution mask.

{/* <ScionaViewerPlaceholder cdgId="solution.kaggle.hubmap_kidney_1st" readOnly={true} /> */}

## When to Use This

- Input images are gigapixel-scale (>10,000x10,000 pixels) and cannot fit in GPU memory whole, requiring tile-based processing
- Histopathology or microscopy images with variable staining intensity across slides/scanners, requiring color normalization for generalization
- Binary or few-class segmentation where IoU is the evaluation metric and a surrogate IoU loss (Lovasz) can directly optimize it
- Target structures (e.g., glomeruli) are small relative to the full image but large enough to be captured within a single tile

## Key Insight

The three-part combination of Macenko stain normalization, Lovasz-BCE loss, and overlapping-tile stitching addresses the three core challenges of gigapixel histopathology: cross-scanner color variation, metric-loss mismatch for IoU evaluation, and memory-constrained inference on images too large to process whole. Each component is individually well-known, but their specific combination is what makes this pipeline robust for gigapixel medical segmentation.

## Pipeline Stages

### Tiling :warning:

Cut the enormous microscope image into small overlapping squares that can fit in memory, so each square can be processed independently.

**Bound atom:** `sciona.atoms.ml.sklearn.images.extract_patches_2d`

### Stain Normalization :white_check_mark:

Adjust the colors of each tissue tile so that all images look like they were stained and scanned the same way, since different labs use slightly different dye concentrations.

**Bound atom:** `sciona.atoms.medical_imaging_3d.preprocessing.macenko_stain_vectors`

### Segmentation Architecture :warning:

Train a segmentation neural network that labels each pixel as glomerulus or background, using a modern backbone architecture with attention mechanisms.

**Bound atom:** `sciona.atoms.dl.architectures.unet_2d_segmentation`

### Lovasz-BCE Loss :warning:

Use a specialized training objective that directly optimizes the competition scoring metric (overlap between predicted and true masks) combined with a standard pixel-by-pixel loss for training stability.

**Bound atom:** `sciona.atoms.dl.loss.lovasz_softmax_loss`

### TTA on Tiles :white_check_mark:

For each tile, predict the segmentation mask on the original plus 7 rotated/flipped versions, then average all predictions to get a more reliable mask.

**Bound atom:** `sciona.atoms.dl.image_augmentation.tta_geometric_average`

### Gigapixel Stitching :warning:

Put all the small predicted masks back together into one giant mask matching the original image size, averaging where tiles overlap, and converting probabilities to yes/no predictions.

**Bound atom:** `sciona.atoms.ml.sklearn.images.reconstruct_from_patches_2d`
