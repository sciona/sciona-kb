## Key Insight

## Pipeline Stages

### Sliding window extraction :red_circle:

This stage segment raw 3D accelerometer data into 2-second overlapping windows (numpy).

### Spectral feature extraction :red_circle:

This stage apply Fast Fourier Transform (FFT) and calculate power spectral densities for each axis (scipy.signal).

### CNN processing :red_circle:

This stage pass the raw sequence chunks through a 1D CNN + Bi-GRU architecture (PyTorch).

### Model ensembling :red_circle:

This stage blend the predictions from the deep learning architecture with a LightGBM model trained on the spectral features.

### Post-processing :red_circle:

This stage apply a Gaussian smoothing filter over the time-series probability predictions to eliminate biologically impossible rapid state flipping.
