# Toxic Comment Classification 1st Place — Back-Translation Augmented BiRNN Ensemble

> Classify online comments into six toxicity categories by generating extra training data through round-trip translation, feeding word embeddings through bidirectional recurrent networks with parallel pooling strategies, and combining multiple models with a gradient boosting meta-learner.

{/* <ScionaViewerPlaceholder cdgId="solution.kaggle.toxic_comment_1st" readOnly={true} /> */}

## When to Use This

- Multi-label text classification with severe per-label class imbalance (some labels &lt;5% prevalence)
- Noisy, user-generated short text (comments, reviews, social media) where spelling and grammar vary widely
- Moderate dataset size (10K-500K samples) where augmentation provides meaningful gains but pre-trained LLMs are not yet dominant
- Multiple correlated labels where a stacking meta-model can learn inter-label dependencies

## Key Insight

Back-translation generates semantically equivalent but lexically diverse paraphrases, which acts as a form of invariance training: the model learns to classify based on meaning rather than surface-level word patterns. This is especially powerful for imbalanced multi-label problems because it selectively upsamples minority labels without duplicating exact examples, avoiding the overfitting trap of naive oversampling.

## Pipeline Stages

### Back-Translation Augmentation :warning:

Create extra training examples by translating comments to other languages and back, producing slightly different wordings of the same meaning, especially for rare toxicity types.

**Bound atom:** `sciona.atoms.dl.back_translation.translate_text`

### Tokenization and Embedding :warning:

Break comments into words, convert each word into a numeric vector using pre-trained word dictionaries (FastText and GloVe), and pad all comments to the same length.

**Bound atom:** `sciona.atoms.ml.tokenizer.tokenize`

### Bidirectional RNN Features :warning:

Process each comment in both forward and backward direction using a recurrent neural network, with random dropout to prevent overfitting, producing a sequence of context-aware representations.

**Bound atom:** `sciona.atoms.dl.architectures.recurrent_sequence_model`

### Dual Pooling :red_circle:

Summarize the variable-length sequence output into a fixed-size vector by taking both the maximum and average values across all positions, then combining them.

### Classification Head :warning:

A final layer that converts the pooled features into 6 probability scores, one for each type of toxicity.

**Bound atom:** `sciona.atoms.dl.architectures.linear_prediction_head`

### Meta-Model Ensemble :warning:

Train a gradient boosting model to combine predictions from many different neural network runs, learning which models to trust more for each type of toxicity.

**Bound atom:** `sciona.atoms.dl.image_augmentation.fold_ensemble_average`
