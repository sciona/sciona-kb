# Gradient_oracle_evaluation

> Computes objective value and gradient for the current parameters using the provided AD mode, while threading RNG and algorithm state as explicit immutable inputs/outputs.

## Conceptual Summary

State-Threaded First-Order Differential Oracle. Evaluates a scalar-valued functional at a candidate point in parameter space and computes its first-order derivative with respect to that point using a selected differentiation strategy. The operation maps between flat and structured parameter representations, writes derivatives into a provided output container, and returns all side-channel context (stochastic key and algorithm state) as explicit immutable outputs. This yields a purity-preserving oracle step suitable for composable iterative methods.. Applications: Calibrating constitutive parameters in computational mechanics by evaluating loss and sensitivity of simulation outputs to model coefficients., Estimating control-policy parameters in robotics using gradient-based updates with explicit simulator and randomness state threading., Fitting inverse-problem parameters in medical imaging reconstruction where objective value and gradient are required per iteration., Optimizing hyperparameters of physics-informed neural operators by computing scalar objective and parameter sensitivities in a reproducible pipeline.
