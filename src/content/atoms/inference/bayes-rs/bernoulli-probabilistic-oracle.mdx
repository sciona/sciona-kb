# Bernoulli_probabilistic_oracle

> Defines and evaluates a Bernoulli distribution as a pure probabilistic oracle with no persistent mutable state.

## Conceptual Summary

Binary_State_LogMass_Oracle. Maps a binary observation and a unit-interval parameter to the log of a two-outcome discrete mass function. The computation is closed-form and elementwise: log_mass = x * log(p) + (1 - x) * log(1 - p), with broadcasting over arrays when provided. It performs no parameter updates or internal caching, so identical inputs always produce identical outputs.. Applications: Digital communications: scoring observed bit sequences under a binary noise/channel model., Reliability engineering: evaluating pass/fail component test outcomes against hypothesized failure propensity., Computational biology: modeling binary assay readouts (activation/inactivation) under a latent success-rate parameter.
