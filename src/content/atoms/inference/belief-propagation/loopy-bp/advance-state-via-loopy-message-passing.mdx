# Advance_state_via_loopy_message_passing

> If `state_in.t &lt; target_iter`, runs synchronous BP updates until `t == target_iter`: for each edge, computes both directed messages (variable->factor and factor->variable), normalizes each message, writes into `msg_new`, then commits `msg_new` into `msg` and increments `t`.
