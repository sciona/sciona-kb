# Targetlogkerneloracle

> Stateless oracle that evaluates target log-kernel values for candidate/current states used by MH acceptance logic.

## Conceptual Summary

TemperedLogMeasureEvaluator. Maps a single state element from an arbitrary measurable space and a scalar scaling parameter to a real-valued log score representing that state's compatibility with a target measure. The operation evaluates an unnormalized log-kernel (equivalently, negative energy up to constants) at one point, with the scaling parameter modulating concentration/dispersion of the score landscape. It performs no state transition or normalization; it provides a pointwise scalar used by downstream stochastic decision rules.. Applications: Molecular conformation sampling: score candidate geometries via tempered log-potential before accept/reject moves., Bayesian inverse problems in geophysics: evaluate tempered log-posterior for candidate field configurations during stochastic exploration., Robotics trajectory inference: assign tempered log-likelihood to candidate latent motion states in sequential Monte Carlo or MCMC pipelines., Generative model latent-space sampling: compute tempered log-density of candidate latent vectors for transition acceptance criteria.
