# Linearize_dynamics

> Compute Jacobian of system dynamics with respect to state.

## Conceptual Summary

Local Tangent Operator Computation. Evaluates a vector-valued transition function at a specific point and computes its partial derivatives with respect to the primary state coordinates. The result is a first-order linear operator that approximates how infinitesimal perturbations in the input state propagate to perturbations in the next state. This is a local sensitivity map on the tangent space of the original nonlinear transform.. Applications: Stability and sensitivity analysis of biochemical reaction-network models near an operating point, Linearized error propagation in aerospace guidance and navigation estimators, Small-signal analysis of electric power network dynamics around nominal conditions, Local behavior analysis of ecological interaction models near equilibrium states
