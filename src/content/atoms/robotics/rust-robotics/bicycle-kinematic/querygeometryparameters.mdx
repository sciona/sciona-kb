# Querygeometryparameters

> Project front length, rear length, and derived wheelbase from the immutable model spec.

## Conceptual Summary

Bipartite Scalar Projection with Additive Extent Derivation. The operation reads two non-negative scalar components from an immutable structured input and returns them directly as explicit outputs. It then computes a third scalar by linear addition of those components, producing a total 1D extent. No state is changed, and the mapping is fully deterministic..
