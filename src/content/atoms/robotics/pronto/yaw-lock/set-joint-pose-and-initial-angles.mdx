# Set_joint_pose_and_initial_angles

> Store joint identity/position inputs and corresponding initial-angle snapshot as a new immutable state.

## Conceptual Summary

Immutable Indexed-Array State Replacement. Creates a new immutable state object by replacing three aligned fields: an identifier sequence, a numeric coordinate/value sequence, and an initialization snapshot sequence. The transform performs a structural copy with targeted field substitution, preserving all non-target fields from the input state. It enforces index-wise correspondence constraints between identifier and numeric sequences to maintain referential integrity.. Applications: Computational neuroscience: update an immutable model state with channel identifiers, current signal amplitudes, and baseline calibration vector., Distributed systems monitoring: store aligned metric keys, latest measurements, and startup reference readings as a new snapshot state., Geospatial analytics: replace region index labels, current scalar field samples, and initial reference field in an immutable simulation step record., Additive manufacturing control: persist actuator index map, present control values, and initial tuning profile as a versioned machine state.
