# Read_initial_joint_angles

> Read stored initial joint angles from immutable state.

## Conceptual Summary

Immutable Baseline Vector Retrieval. Consumes an immutable structured state object and performs a schema-bound field projection to extract a predefined numeric collection. The operation applies no numerical transformation and does not mutate input state; it returns the stored baseline sequence exactly as represented (by value copy or read-only reference). The mapping is deterministic and repeatable for identical input state.. Applications: Computational neuroscience: retrieve initial parameter vectors from immutable simulation snapshots before iterative integration, Materials modeling: extract baseline coefficient arrays from frozen experiment configurations for reproducible solver initialization, Distributed optimization systems: read starting weight vectors from immutable checkpoints to ensure restart determinism
