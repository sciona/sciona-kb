# Set_standing_link_targets

> Set left/right standing link identifiers in a newly returned immutable state.

## Conceptual Summary

Immutable Dual-Reference Field Substitution. Takes an immutable structured state and performs a selective substitution of exactly two designated reference-valued fields using externally provided identifiers. It returns a new state instance with those two fields overwritten while preserving all other fields bit-for-bit from the input structure. The operation is a non-iterative, deterministic structural copy-with-update transform.. Applications: Versioned graph analytics: replace predecessor/successor node identifiers in a snapshot without mutating prior graph states, Distributed systems configuration: update primary/secondary endpoint IDs in an immutable failover configuration object, Computational geometry or CAD: swap two boundary-connection handles in an immutable mesh/topology state, Molecular simulation pipelines: revise two bonded-partner indices in an immutable interaction-state record
