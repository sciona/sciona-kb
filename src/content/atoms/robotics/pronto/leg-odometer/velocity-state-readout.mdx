# Velocity_state_readout

> Reads immutable velocity state-space components (body-frame velocity and its covariance) and returns the current velocity estimate.

## Conceptual Summary

Immutable Latent Moment Readout. Takes an immutable state container holding the first-order moment (mean vector) and second-order uncertainty structure (covariance matrix) of a latent variable, then performs a direct readout of those components. No inference, filtering, or parameter update is applied; the output is an exact structural projection of stored moments.. Applications: Neuroscience signal processing: read current latent neural activity estimate and posterior covariance from a state-space decoder, Climate data assimilation: extract present atmospheric/ocean latent state mean and uncertainty from an analysis snapshot, Industrial process control: retrieve estimated hidden process variables and confidence structure from a digital twin state, Network operations engineering: read inferred traffic-flow state vector and covariance from a probabilistic monitoring model

## Type Signature

```
(state_in: LegOdometerState{xd_b_, vel_cov_}) -> tuple[Vector-like velocity estimate, Covariance matrix]
```
