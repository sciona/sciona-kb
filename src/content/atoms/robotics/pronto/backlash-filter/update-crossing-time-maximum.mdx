# Update_crossing_time_maximum

> Produce a new filter state with an updated maximum crossing time.

## Conceptual Summary

ImmutableStateScalarBoundUpdate. Applies a single-field substitution on an immutable state object: one designated scalar threshold field is overwritten with a provided finite scalar, while all other fields are preserved exactly. This is a deterministic state-transition operator that returns a new state instance rather than mutating the input. The transform changes only the selected parameter and leaves the structural schema invariant.. Applications: Control systems: updating a saturation or timeout bound in a controller parameter state without altering other controller coefficients., Distributed systems: replacing a node-level heartbeat deadline threshold in an immutable runtime configuration snapshot., Computational biology: updating a maximum iteration or convergence cutoff parameter in a simulation state object., Robotics/perception: revising an outlier-rejection threshold inside a filter state while preserving accumulated latent variables.
