# Detect_peak_candidates

> Scans the normalized prediction using a sliding window (size 15, or 7 near the end) to locate candidate peak indices. For each window where the threshold ≥ 0.70 is met, finds the position of the maximum normalized value; if ties exist, resolves them by selecting the position of the maximum in the corresponding raw signal window. Advances the cursor by the window size after each match, or by 1 otherwise.
