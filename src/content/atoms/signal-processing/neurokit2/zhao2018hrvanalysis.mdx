# Zhao2018hrvanalysis

> Applies Zhao et al. 2018 method to analyze heart rate variability from ECG signals using windowed processing

## Conceptual Summary

windowed_inter_event_variability_extractor. Converts a 1D continuous signal and a sequence of discrete event markers into a set of statistical variability features by first computing inter-event temporal intervals, then applying windowed segmentation and extracting distributional moments, frequency-domain characteristics, and entropy measures within each segment. This is a dimensionality-reducing lossy transform from a time-indexed event sequence to a compact feature vector characterizing temporal regularity/irregularity.. Applications: Seismology: Characterize earthquake clustering patterns by analyzing variability in inter-event times between seismic events within moving time windows to distinguish Poissonian from correlated seismicity., Network traffic engineering: Detect congestion, DDoS attacks, or protocol anomalies by computing variability features from packet or connection inter-arrival times over sliding windows., Manufacturing quality control: Monitor machine health by tracking variability in cycle completion intervals or defect occurrence times, flagging increased irregularity as early wear indicators., Neuroscience: Classify neuronal firing patterns (bursting vs. regular spiking) by extracting inter-spike interval variability features from electrophysiology recordings in sliding temporal windows.
