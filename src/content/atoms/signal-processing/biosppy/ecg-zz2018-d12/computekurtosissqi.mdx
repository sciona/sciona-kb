# Computekurtosissqi

> Computes kurtosis-based ECG quality score from the input signal.

## Conceptual Summary

Standardized Fourth-Moment Signal Concentration Score. Maps a 1-D sampled scalar sequence to a single distribution-shape statistic by computing its standardized fourth central moment. The transform quantifies tail heaviness and peak concentration relative to variance, producing either excess kurtosis (Fisher convention) or raw kurtosis (Pearson convention) based on a binary mode selector. This is a many-to-one statistical reduction from sample-level structure to a scalar quality/regularity indicator.. Applications: Mechanical condition monitoring: score impulsiveness of accelerometer streams to detect bearing or gear faults., Communications engineering: characterize burstiness/non-Gaussianity of channel noise or interference samples., Geophysical sensing: detect transient event contamination in seismic trace windows via tail-heaviness scoring., Neurosignal processing: assess artifact contamination in electrophysiology segments using fourth-moment concentration.
