# Iterate_pdb_residues

> Iterates over all residues in the PDB structure. An optional filter can be applied to return only residues that contain a specified chemical element.

## Conceptual Summary

query_substructures_by_component_property. Traverses a hierarchical data structure composed of nested substructures, which in turn contain primitive components. It generates a sequential, lazy iterator over the substructures, with an optional filter to include only those that contain a primitive component matching a specific categorical property.. Applications: In a 3D scene graph, iterating over all mesh groups that contain a specific primitive type, like a light source or a collision object., In a file system representation, finding all directories that contain a file with a specific extension (e.g., '.log')., In a parsed XML/HTML document, yielding all parent nodes (e.g., '&lt;div>' tags) that contain a child node with a specific attribute., In a logistics database, iterating through all shipping containers that are registered to be carrying a specific class of item (e.g., 'refrigerated goods').
