# Tt2tdb_offset

> Computes the relativistic time scale offset between Terrestrial Time (TT) and Barycentric Dynamical Time (TDB) in seconds, using a sinusoidal approximation of the periodic correction term derived from Earth's orbital eccentricity.

## Conceptual Summary

PeriodicPhaseOffsetFromElapsedScalar. Maps a monotonically increasing scalar (elapsed time or phase accumulator) to a bounded periodic correction term by evaluating a sinusoidal function whose argument is a linear function of the input.
