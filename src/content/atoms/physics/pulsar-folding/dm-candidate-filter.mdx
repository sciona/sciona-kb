# Dm_candidate_filter

> Evaluates and filters dispersion measure (DM) candidates by comparing observed data against a base DM model, using sensitivity, channel frequency, bandwidth, and time sampling parameters to identify viable DM candidates.

## Conceptual Summary

threshold_gated_reference_deviation_filter. Given an observed multi-dimensional signal array and a reference baseline array, this operation evaluates each element of an input candidate set by computing whether the observed signal's deviation from the baseline — modulated by a set of spectral-temporal resolution parameters (channel bandwidth, center frequency, and sampling interval) — exceeds a scalar sensitivity threshold. Candidates whose derived signal response falls below the threshold are rejected, yielding a pruned subset. The transform is a parametric, discriminative sieve: it maps a candidate index space through a physics-derived scoring function and returns only those candidates satisfying the inequality constraint.. Applications: Gravitational wave astronomy: filtering trial coalescence-time or chirp-mass candidates by comparing observed strain time-series against a waveform template baseline, gated by detector frequency band and sampling rate sensitivity — retaining only candidates whose SNR exceeds a detection threshold, Medical neuroimaging (EEG/MEG source localisation): evaluating a set of candidate dipole source locations by comparing modelled field maps against observed sensor arrays, using electrode bandwidth and temporal resolution to gate candidates below a signal-to-noise sensitivity bound, Radar / sonar signal processing: pruning a set of candidate target range-velocity hypotheses by comparing measured range-Doppler returns against a clutter baseline, with sensitivity scaled by pulse bandwidth and sampling interval — discarding sub-threshold false-alarm candidates, Seismology: screening candidate epicentre locations or fault-rupture depths by evaluating residual misfit between observed waveform envelopes and a reference Earth-model prediction, with threshold gating modulated by seismometer frequency response and sampling period
