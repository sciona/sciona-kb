# Find_fof_clusters

> Computes friends-of-friends (FOF) clusters for a set of points on a periodic grid. This is a core clustering algorithm used in cosmology and astrophysics to identify gravitationally bound structures.

## Conceptual Summary

transitive_proximity_partitioning. Partitions a set of points in an N-dimensional space with periodic boundaries into disjoint subsets. The partitioning is based on constructing a proximity graph where an edge exists between any two points if their distance is below a specified threshold. Each output subset represents a connected component of this graph, effectively forming a transitive closure over the 'is-a-neighbor' relationship.. Applications: Molecular Dynamics: Identifying groups of interacting atoms or molecules in a simulation with a periodic solvent box to study aggregation or complex formation., Epidemiology: Modeling contagion clusters in a population distributed over a geographical area with wrap-around boundaries, where the linking length represents the transmission radius., Materials Science: Analyzing the formation of crystal grains or the propagation of defects within a simulated material lattice that uses periodic boundary conditions., Sensor Networks: Grouping signals from a distributed network of sensors deployed in a toroidal or wrap-around physical space to identify event regions.
